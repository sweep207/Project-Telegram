## 📥 Cài Đặt  

### 1.Lấy Token Bot Telegram  
- Vào Telegram, tìm **BotFather**  
- Gửi lệnh `/newbot` và làm theo hướng dẫn  
- Sao chép **token** bot và lưu lại  

### 2.Clone Repo  
```bash
git clone https://github.com/sweep207/Project-Telegram.git
cd Project-Telegram
```
